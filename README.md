# Proyecto Kira: Chasis Biomimético V1.0
**Estatus:** Desarrollo de Ingeniería Híbrida (Aluminio/3D)
**Arquitectura:** Sistema de tendones accionado por motores paso a paso.
**Red:** LAN aislada (Air-gapped) con procesamiento local mediante ROS 2.

## Estructura de Carpetas
- `/hardware`: Scripts CAD (Python) y diseños de articulaciones.
- `/brain`: Lógica de IA local y parámetros de personalidad.
- `/network`: Protocolos de comunicación segura.
- `/firmware`: Código para controladores de motores.
